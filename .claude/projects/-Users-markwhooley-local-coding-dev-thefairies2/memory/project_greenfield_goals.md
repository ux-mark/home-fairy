The user wants to rebuild The Fairies home automation system from scratch (greenfield) with these priorities:

1. **Mobile-first** design — primary interface is phone
2. **Desktop** support — also works well on larger screens
3. **iWatch** view — critical controls available on Apple Watch
4. **Modern UI/UX** — more efficient and delightful user experience
5. **Efficiency** — more efficient architecture than the current system

**Why:** Current system is functional but the UX is basic (LitElement 2.x, w3.css, file-based routing, no SPA). The user wants a significantly better experience.

**How to apply:** Phase 1 is documentation (in progress). Phase 2 will be greenfield build with modern stack choices informed by the documentation.
